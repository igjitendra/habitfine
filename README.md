# HabitFine Website

Static, responsive website. No build command is required. Open `index.html` or deploy the folder to Netlify, Vercel, GitHub Pages, Cloudflare Pages, or standard hosting.

## Replace screenshots
Put final PNG screenshots in `images/` using these exact names:
- `screenshot-home.png`
- `screenshot-profile.png`
- `screenshot-report.png`
Recommended portrait ratio: 9:19.5 and width at least 1080px.

## Before publishing
1. Replace `YOUR-DOMAIN.example` in `robots.txt` and `sitemap.xml`.
2. Replace the coming-soon Google Play button with your real listing URL.
3. Confirm operator/business name, jurisdiction, support email and subscription details.
4. Have legal pages reviewed by a qualified professional for your jurisdiction.
5. If you add analytics, cookies, embedded video, forms, or ad pixels, update the Cookie and Privacy policies and add consent where required.
